使用 Inline Bot 并通过 EFB 发送微信小黄脸  [draft]

前言 我的微信很早之前就不能使用网页登录了，也因此不能继续使用 EFB, 直到最近听说 EFB 更新了一个 UOS 补丁，可以让部分网页版微信被 ban 的用户重新用上 EFB，于是我也趁此机会重试了一下，安装最新版后发现果然可以，真是十分惊喜。 重新开始使用 EFB，还是有很多不熟悉的地方的，且 bot 的指令较以往有所改动，基本是要重新适应的。使用几天下来后，我发现 EFB 将微信小黄脸转换成 emoji 的样式让我不太习惯，也不太喜欢，部分小黄脸对应的 emoji 更是让我觉得毫不相关，且 EFB 不支持用户发送微信小黄脸，于是我萌生了自己写一个 inline bot 来实现此功能的想法。 测试阶段 首先，我使用之前写过的古诗文助手bot的 inline 模式，通过 EFB 向微信发送消息，发现微信能正确接收到我所发的信息, 且不需要在 efb 的 config 文件的 admin_id 中添加该 bot 的 id，证明通过 inline 模式发送文本这个方法是可行的。当天晚上，我就写了一个 demo 运行了起来, 但是只支持常用的 [旺柴] 和 [捂脸]: 使用方法改进 从第一版到现在正式使用的版本之间有两版中间版本,
