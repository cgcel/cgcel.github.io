恢复备份

按照上述步骤配置好 「用户资料同步」 后, 我通过坚果云备份了生成的用户备份资料, 该资料存储路径被我定义在 D:/NutsBackup/RimeSync/ 中了.

在新的一台电脑安装好 「Rime 小狼毫」 后, 启用坚果云, 并将原先备份好的「用户资料同步」数据同步到本机相同的路径 D:/NutsBackup/RimeSync/, 然后将其中的 .yaml 文件覆盖至用户文件夹中；然后, 打开 Rime 的「用户词典管理」, 选择 luna_pinyin_simp 词典, 然后选择「合入词典快照」, 选择坚果云备份中想要合并的 .userdb.txt 文件, 将其合并至用户词典中.

userdb 中的一些参数意义

c = 词频 d = 权值 t = 时刻

用户词典的词条在每次输入的时候会更新, t 记录的是他最近一次更新的输入法时间. 至于词频 c, 一年前输入的一次和刚刚输入过的一次其表现出来的效果肯定不一样, 所以会综合时间参数 t 折算成一个权值 d, d 值并不是当前的权重, 而是 t 时刻该词的权重. 随着时间流逝, d 值会衰减的.

不要被用户词库里自己的创造吓到, 如果有心维护一下, 不会正则表达式还是算了, 毕竟条目太多了, 而且大多数时候不影响你的使用. 不过前几天网友的案例倒是一个提醒, 他反馈的问题现象是输入某些特定字母时输入法卡顿, 找了一圈原因, 最后发现是用户词库里有一堆不恰当的首字母分词导致的, 因为当我们输入字母时, 输入法会基于词库进行检索和匹配响应, 原本应该是shu ru fa, 结果词库里有s hu那么判断关联到的就是不同的词汇, 这种非元音的不合理切分也会导致拼音组合混乱, 检索量级上来了就表现出卡顿现象.

多设备词库覆盖（合并）原理

.userdb 是增量合并的, 那么如果你修改了 .userdb.txt, 该如何部署到输入法里呢？

  • 如果你有多台电脑的同步配置, .userdb.txt 文件修改好以后先覆盖另一个同步文件夹里的同名 .userdb.txt 文件；
  • 删除「用户设定」也就是本地配置文件夹下的 同名 .userdb 文件夹；
  • 菜单栏执行「同步用户数据」, 输入法会基于 .userdb.txt 生成新的 .userdb 文件夹从而实现用户词库的修改.

中英文混输「easy_en」

Rime 系列输入法自然也可以实现 #中英文混输 , 这对我来说是一项很常用的功能. 当年在 Google 拼音就经常用九宫格「造词」, 打出自己想要的单词让输入法强行记住, 这样可以不用切换至英文输入也能打出简单的英语单词或者自己的造词.

# luna_pinyin_simp.custom.yaml
patch:

  __include: easy_en:/patch
  easy_en/enable_sentence: false

  'engine/filters/@before 0':
    simplifier@emoji_suggestion
  emoji_suggestion:
    opencc_config: emoji.json
    option_name: emoji_suggestion
    tips: all

      - { when: composing,  accept: Control+c,  send: Escape } # ctrl-c 清屏
      - { when: composing,  accept: -,  send: Page_Up } # 候选词分页
      - { when: composing,  accept: =,  send: Page_Down } # 候选词分页
      - { when: always,  accept: Control+period,  toggle: ascii_punct } # 切换西文/中文标点 Ctrl + .

小鹤双拼配置过程

在配置了简体全拼的基础上, 小鹤双拼的配置会变得非常简单.

下载安装

首先我们下载官方提供的双拼输入方案, 将 double_pinyin_flypy.schema.yaml 也即我们需要安装的小鹤双拼方案, 放置在用户目录中, 重新部署即可完成安装.

启用简体小鹤双拼

通过任务栏右下角图标或快捷键进入 RIME 设置, 选择小鹤双拼后根据使用习惯选择简体字输入, 如此一来就设置好了 PC 端的小鹤双拼了.

加载自定义配置

当然, 只设置好了输入方案是不够的, 还需要设置用户自定义的词库、输入习惯等项才能称为一个合格的 RIME 输入法.

按照 RIME 命名规范, 在用户目录创建 double_pinyin_flypy.custom.yaml, 意为小鹤双拼的用户自定义配置文件, 然后进行以下操作：

雾凇拼音配置过程

配置

参考 雾凇仓库 README, 将 RIME 用户目录下的文件以及文件夹全部清除, 然后进行雾凇的安装.

粤语词典维护

使用雾凇一段时间后发现, 粤语词汇应该是没有包含在雾凇的词典中, 应该是作者和用户没有粤语地区用户, 或是用户自行挂载了粤语词典, 于是打算维护一个粤语词库, 顺便多了解一些粤语词汇, 增长见识.

问题解决

清除缓存

在使用过程中经历了一下场景: 从默认为繁体的 rime_pinyin 输入方案转至 rime-ice 雾凇方案, 会导致候选词中有大量残留的繁体字缓存.

解决方法

  1. 停止小狼毫算法服务
  2. 进入用户目录, 删除 rime_ice.userdb 文件夹下的所有文件
  3. 进入用户词典管理, 将已有的 userdb.txt 中的内容转成简体或去除多余的繁体字的行
  4. 合入词典快照, 选择编辑好的 userdb.txt
