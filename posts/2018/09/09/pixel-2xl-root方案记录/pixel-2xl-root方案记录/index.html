Pixel 2XL root方案记录

开篇

回顾我的刷机历程, 每次都是有了想法之后就行动起来了, 经历的是一次又一次的真香警告. 而本次 root 的想法的起因是黑域越来越难用, 让我想用回绿守, 本想着等 9.1 出来之后再 root, 但是每次我都会等不及… 然后果不其然的踩坑了.

刷机环境以及刷机包准备

  • USB驱动
  • adb环境
  • TWRP
  • Magisk

其中, twrp 较新版本刷入之后有 bug, 上 youtube 看过教程后, 决定采用 twrp3.2.1.2, 下载时需要下载.img以及.zip文件.

具体步骤

  1. 进入命令行

  2. 进入 fastboot 界面

    $ adb reboot bootloader

  3. 解 bootloader 锁

    $ fastboot flashing unlock

    使用音量键和电源键进行确认, 注意解锁后手机所有数据将会被清除, 要做好备份.

  4. 刷入第三方 recovery

    $ fastboot boot path/twrp.img

    刷入成功后, 重启进入 recovery, 此时还需要进入 install, 选择 twrp.zip 包再进行刷入. 缺少这一步的话, 下一次重启还会回到官方recovery.

  5. 刷入 magisk 在 twrp 的 install 页面选择事先存入手机的 magisk.zip 进行刷入, 刷入后重启即可.

结尾

整个过程下来就刷 twrp 出了些问题, 最新版本不能用是什么鬼…? (勘误: 事实上只是 twrp 的加密, 输入锁屏 PIN 码即可) 好在最后还是解决了问题, 顺便写篇博客记录一下这个坑.
